# README

## Program Overview

This Python program contains **two tasks**:

1. **Calculate Factorial Using a Function**

   * Asks the user to enter a number.
   * Uses a **recursive function** to calculate the factorial of that number.
   * Prints the factorial result.

2. **Using the Math Module for Calculations**

   * Asks the user to enter a number (float allowed).
   * Performs the following calculations using the `math` module:

     * **Square root**
     * **Natural logarithm (base e)**
     * **Sine value (in radians)**
   * Displays the results.

---

## How to Run

1. Save the code in a file (e.g., `main.py`).
2. Open a terminal or command prompt.
3. Navigate to the folder where the file is saved.
4. Run the program with:

```
python main.py
```
